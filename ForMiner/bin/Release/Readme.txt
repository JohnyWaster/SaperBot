Бот для сапера
1. После запуска приложение прячется в трей, разверните его.
2. После нажатия на кнопку старт через некоторое время начнется игра.
3. Если игра не началась, значит в вашей системе сапер лежит не в том месте, откройте его самостоятельно
и перезапустите приложение.
4. Размер окна сапера должен быть стандартным, то есть минимально возможным.
5. При нажатии на любую клавишу игра приостановится, для продолжения можно снова нажать любую клавишу, либо кнопку старт.
6. При нажатии на стоп игра остановится, статистика спрячется, для возобновления нажмите старт.
7. При выставлении галочки "добавить в автозагрузку" игра добавится в автозагрузку, если позволит система.
8. При выставлении "использовать как заставку" и установки времени больше 0, игра будет автоматически стартовать
при бездействии клавиатуры и мыши заданное число минут.
9. Скорость игры тем выше, чем левее ползунок скорости.
